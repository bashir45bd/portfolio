# 🚀 Mobile Developer Portfolio Website

A modern, responsive and production-style **portfolio website** built using HTML, CSS, Bootstrap 5, and JavaScript.  
This project is designed to showcase **mobile application development skills, projects, experience, and technical expertise** in a clean and professional UI.

---

# 📌 Project Overview

This portfolio website is a **static front-end project** that demonstrates modern UI/UX design practices and developer-level frontend engineering structure.

It is built to represent a real-world developer profile including:

- Project showcase system
- Technical skills visualization
- Work experience timeline
- Blog/article system (static)
- Contact section UI
